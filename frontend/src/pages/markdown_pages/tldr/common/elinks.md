# elinks

> A text based browser similar to `lynx`.
> More information: <https://github.com/rkd77/elinks>.

- Start ELinks:

`elinks`

- Quit elinks:

`<Ctrl c>`

- Dump output of webpage to console, colorizing the text with ANSI control codes:

`elinks -dump -dump-color-mode {{1}} {{url}}`
