# mariadb-install-db

> Initialize a MariaDB database.
> More information: <https://manned.org/mariadb-install-db>.

- Initialize a database:

`sudo mariadb-install-db --user {{user}} --basedir {{/usr}} --datadir {{/var/lib/mysql}}`

- Display help:

`mariadb-install-db --help`
