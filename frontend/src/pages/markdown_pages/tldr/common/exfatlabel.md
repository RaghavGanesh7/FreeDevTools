# exfatlabel

> Get or set an exFAT filesystem label.
> More information: <https://manned.org/exfatlabel>.

- Display the current filesystem label:

`exfatlabel {{/dev/sda}}`

- Set the filesystem label:

`exfatlabel {{/dev/sda}} {{new_label}}`
