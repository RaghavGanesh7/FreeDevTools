# gouldtoppm

> Convert a Gould scanner file to a PPM image.
> More information: <https://netpbm.sourceforge.net/doc/gouldtoppm.html>.

- Convert a Gould scanner file to a PPM image:

`gouldtoppm {{path/to/file.gould}} > {{path/to/output.ppm}}`
