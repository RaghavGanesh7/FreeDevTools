# d8

> Developer shell for the V8 JavaScript engine.
> More information: <https://v8.dev/docs/d8>.

- Start a REPL (interactive shell):

`d8`

- Run a JavaScript file:

`d8 {{path/to/file.js}}`

- Evaluate a JavaScript expression:

`d8 -e "{{code}}"`
