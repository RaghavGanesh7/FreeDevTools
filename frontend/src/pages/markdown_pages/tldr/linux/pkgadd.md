# pkgadd

> Add a package to a CRUX system.
> More information: <https://docs.oracle.com/cd/E88353_01/html/E72487/pkgadd-8.html>.

- Install a local software package:

`pkgadd {{package}}`

- Update an already installed package from a local package:

`pkgadd -u {{package}}`
