# vgdisplay

> Display information about Logical Volume Manager (LVM) volume groups.
> See also: `lvm`.
> More information: <https://manned.org/vgdisplay>.

- Display information about all volume groups:

`sudo vgdisplay`

- Display information about volume group vg1:

`sudo vgdisplay {{vg1}}`
