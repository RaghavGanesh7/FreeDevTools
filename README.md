## Quick Start

1. **Clone the repository**

   ```bash
   git clone https://github.com/yourusername/freedevtools.git
   cd freedevtools
   ```

2. **Install dependencies**

   ```bash
   cd frontend
   npm install
   ```

3. **Run development server**

   ```bash
   make run
   ```

4. **Open your browser**
   Navigate to `http://localhost:4321`
